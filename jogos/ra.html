Coloque o tabuleiro no centro da mesa.
Coloque todos os fichas de jogo no saco.
Cada jogador pega 10 pontos em fichas de pontos de vitória (VP). Coloque o resto das fichas perto do tabuleiro.

Baseado no número de jogadores, ordene as fichas de sol em grupos da seguinte forma:
2 jogadores: 9-6-5-2 ; 8-7-4-3
3 jogadores: 13-8-5-2 ; 12-9-6-3 ; 11-10-7-4
4 jogadores: 13-6-2 ; 12-7-3 ; 11-8-4 ; 10-9-5
5 jogadores: 16-7-2 ; 15-8-3 ; 14-9-4 ; 13-10-5 ; 12-11-6

Coloque a figura do Ra perto do tabuleiro.
O jogador com a maior soma de sóis começa; continue no sentido do relógio.

No turno do jogador, ele pode fazer exatamente uma das seguintes ações:
Comprar e colocar uma ficha no tabuleiro
Jogar uma ficha de Deus (fichas com fundo amarelo)
Invocar o Ra

Um jogador sem sóis não pode realizar nenhuma ação no seu turno. Quando a trilha de leilão está cheia, jogadores estão limitados a apenas jogar fichas de Deus ou Invocar Ra.
Quando uma ficha de Deus é jogada por um jogador, a ficha de Deus vai para a caixa.
uma ficha de Deus não pode ser usado para pegar um outro tile de Deus e/ou uma ficha de Ra
Quando uma ficha de Deus é utilizado para pegar uma ficha de desastre, o jogador deve colocar o tile de desastre em sua área de ação e realizar imediamentente o efeito do tile de desastre.
uma ficha de Deus só pode ser jogado por um jogador enquanto ele ainda possui sóis virados para cima.
Se todos os 8 fichas de leilão já estiverem no tabuleiro, um jogador não pode comprar e nem colocar uma ficha. As únicas ações para esse jogador são jogar uma ficha de Deus ou invocar Ra.
Se a trilha de fichas de leilão do tabuleiro estiver completa (oito fichas) e o jogador escolher invocar Ra, TODOS os jogadores (inclusive o jogador que ativou o Ra) podem dar lances ou podem PASSAR. Se todos os jogadores passarem, os fichas são remo vidos do jogo.
Quando a trilha de fichas de Ra está completa, não há leilão. Neste caso, os fichas na trilha de leilão são removidos e retornados para a caixa.
Quando um leilão é concluído, o jogador à esquerda do do jogador com o Ra continua o jogo fazendo um turno normal.
Quando um Ra é invocado ou uma ficha de Ra é retirado do saco, o jogador deve pegar a figura de Ra e colocar na frente dele e o leilão começará com o próximo jogador da esquerda ao jogador com a figura de Ra.
Em um leilão, normalmente um jogador pode passar. Entretanto, se o leilão foi invocado voluntariamente por por algum jogador, esse jogador DEVE obrigatoriamente fazer um lance se todos os outros jogadores passarem.
fichas removidos do jogo (que vão para a caixa) não podem ser examinados pelos jogadores.
No caso de um leilão involuntário, todos os jogadores podem passar. Nesse caso, todos os fichas na trilha de leilão são descartados e voltam para a caixa.
Ao final de cada uma das três épocas, os jogadores recebem VP e mantêm seus VP virados para baixo.

Deuses


Cada ficha de Deus em jogo dá 2 pontos (VP) para o jogador que a possui.
A ficha volta para a caixa.

Faraós


O jogador com mais fichas de faraós recebem 5 VP e jogador com menos fichas de faraós perde 2 pontos. Se dois ou mais jogadores empatarem com mais ou menos fichas, cada jogador (empatantes) ganha 5 VP ou 2 VP. Se todos os jogadores possum o mesmo número de fichas de faraós, pontos não são perdidos e nem ganhos.
As fichas NÃO voltam para a caixa.

Nilo


Cada tile de Nilo e cada tile de Nilo inundado valem 1 VP somente se o jogador possuir pelo menos 1 tile de Nilo inundado.
As fichas de Nilo inundados voltam para caixa e os fichas de Nilo NÃO voltam para a caixa.

Civilização


Após cada época, se o jogador NÃO possui fichas de civilização, ele perde 5 VP. Se o jogador possui 3/4/5 fichas DIFERENTES, ele ganha 5/10/15 VP.
Todos as fichas de civilização voltam para caixa.

Ouro


Cada tile vale 3 VP.
As fichas voltam para a caixa.

Monumentos


Os fichas de monumentos são pontuados SOMENTE na terceira época. Se o jogador possuir 1-6/7/8 fichas de monumentos DIFERENTES, ele ganha 1-6/10/15 VP. E, se o jogador possuir 3/4/5 fichas de monumentos IGUAIS, ele ganha 5/10/15 VP.

Sóis

Sóis são pontuados SOMENTE na terceira época. Cada jogador soma os números dos seus sóis (ambos virados para baixo e para cima). O jogador com a maior soma ganha 5 pontos e o jogador com a menor soma perde 5 pontos.
